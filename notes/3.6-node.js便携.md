node.exe便携版本，没有npx
需求：跨电脑运行，优先使用内置的npx,不用用户安装
解决方案：
下载 npm 包到 bin 目录，用 node.exe 执行 npx-cli.js。
mkdir -p bin/npm; cd bin/npm; python -c "import urllib.request, tarfile, io; url='https://registry.npmjs.org/npm/-/npm-11.11.0.tgz'; data=urllib.request.urlopen(url).read(); tar=tarfile.open(fileobj=io.BytesIO(data)); tar.extractall('.'); print('下载完成')"
